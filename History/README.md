# History Folder Rule

This folder contains only intentionally retained source-history material.

Canonical updates are edited in place. A downloaded repository version is not copied into the next repository as a pre-edit snapshot. Routine changes are recorded concisely in `CHANGELOG.md`. Full prior-file copies are kept only when the author explicitly requests them or when a source is independently significant rather than merely an older revision.

Pre-audit mass snapshots are intentionally excluded from the distributed master because they contain a false transport-to-mass assumption that repeatedly contaminated AI summaries. The original uploaded archives remain the only external provenance copy.
