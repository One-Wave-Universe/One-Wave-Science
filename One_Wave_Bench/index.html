How this bench works

This is a turn-based workbench. Either human or ai proposes an experiment as a request (hypothesis + parameter cases + falsifiers). The runner evolves the D-413 state, computes D-412 measurements, runs control/failure checks, and writes a receipt. Both parties read the same receipts here.

D-412 discipline is enforced: state → measured fields → visualization. A run can fail. Interpretation is kept separate from raw output. Nothing here claims gravity, charge, a proton, a Mirror Gate, or Mass Effect — Stage 01 is a Yellow reduced model.
