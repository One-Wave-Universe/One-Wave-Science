# Node B-202: Pressure

Dependencies:
Upstream: B-201 Equilibrium Balance
Downstream: Force Density, Local Stability, Bounded Motion (cross-ref A-107), B-206 Paired Loop

IMPORTANT: E-503 Pressure (Gradient Form) in Appendix E is a downstream specialization of
A-104 Gradient, NOT of this node. These are distinct nodes. Do not merge.

Definition:
Pressure defines the local state generated by imbalance within the One-Wave field.
Pressure is not itself a force and does not describe motion.
Pressure is a field quantity, not a surface quantity.

P = f(B)

where f maps local imbalance into a pressure state.

Mathematics:
If B = 0 and f(0) = 0: P = 0
If B != 0: P = f(B)

Imbalance => Pressure

The constitutive form of f remains unspecified.

Operational Chain:
Balance => Pressure => Force Density => Response
