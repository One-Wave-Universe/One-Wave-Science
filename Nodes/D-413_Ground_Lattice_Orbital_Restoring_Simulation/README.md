# D-413 Ground Lattice Orbital-Restoring Simulation

Open `index.html` in a modern browser for the canonical D-413 simulation.

Open `ai_human_workstation.html` inside the Codex visualization surface for the
combined AI/human experiment workstation. Its simulation and CSV export also
work as a standalone file; AI dialogue requires a host integration and contains
no hard-coded API key.

The simulation renders a native-2D triangular Ground as a visibly deformed curvature surface. The shaded triangular faces, curved contours, well-depth marker, bounded displacement height, and restoring-gradient calculation all use the same evolving lattice state. `Vertical exaggeration` is display-only. Camera and projection changes never alter the physics.

Controls include Ground-fixed, displacement-fixed, and well-fixed cameras; curved-surface and top-down projections; curvature depth/radius; lattice response; shell asymmetry; torque coupling; overlays; CSV export; and control/ablation presets.

The AI/human workstation adds a scrolling displacement view, paired Field/Void
motion through Common Ground, the twelve-state cycle, `Theta=(q,a,p)`
thresholds, hysteretic gates, resistance, surface tension, internal
pressure/voltage, experiment presets, CSV receipts, wiki requests, and
Codex-hosted experiment dialogue.

The curvature well is imposed and the bounded shell is a Yellow reduced coordinate. This does not derive gravity, quarks, protons, charge, Mirror Gate, or Mass Effect.
