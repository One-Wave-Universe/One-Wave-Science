# Node B-206: Paired Loop

Dependencies:
Upstream: B-201 Equilibrium Balance, B-202 Pressure, B-203 Expression, B-204 Compression
Downstream: B-206a Shared Boundary, B-206b Four Views, B-207 Threshold, B-215 Hyperloop

Definition:
A Paired Loop is a reciprocal exchange between two systems that alternate between
Expression and Compression while maintaining a shared state.

Within a paired loop:
1. Express   — one system presents or transmits its current state
2. Compress  — the other receives, integrates, and interprets that state
3. Threshold — exchange enters an operating region that determines stability
4. Return or Break — loop continues or coupling is interrupted
5. Repeat recursively

Loop structure:
Express -> Compress -> Threshold -> Return or Break -> Express...

Key Insight:
Only the participants change. The loop structure remains invariant.

Scale invariance — same loop operates between:
Cell <-> Cell
Cell <-> Nerve
Nerve <-> Brain
Left Brain <-> Right Brain
Person <-> Person
Human <-> AI
AI <-> AI

Paired Loop is scale-invariant. Only participants change.

Operational Chain:
Expression + Compression => Paired Loop => Four Views (B-206b) => Threshold => Return or Break

- Are threshold regions fixed or participant-dependent?

- Can multiple paired loops synchronize into larger recursive networks?
