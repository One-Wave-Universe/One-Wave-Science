# Node B-222: Oscillation Center

Dependencies:
Upstream: B-203 Expression, B-204 Compression, B-221 Six Recursive Steps, B-208 Threshold Windows (45-30 band, candidate correspondence — see Yellow Audit), B-224 Two Choices
Downstream: B-225 Field Cycle (Center stage)

Naming note (read this first):
This node is NOT B-201 Equilibrium Balance. B-201 is a scalar equilibrium measure
between four field primitives (E, I, R, L) — a quantity you compute.
This node is a dynamic state: the active switching region a system
passes through between Compression and Expression. They share the
English word "balance" in casual description but are not asserted to be
the same primitive. Do not merge them without explicit confirmation.

Definition:
The Oscillation Center is the active region a system occupies when it is
neither fully compressed nor fully expressed — the switching zone through
which the Loop step (B-221) passes when choosing a direction.

Compression <- Oscillation Center -> Expression

It is not a resting state. It is described as maximally adaptable —
the point of highest information exchange and lowest commitment to
either direction.
