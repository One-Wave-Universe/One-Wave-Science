# Node A-105: RESTORING RESPONSE

LAYER: Response Layer

TERMINOLOGY CORRECTION (this session): the symbol was previously
F_OW ("Force, One-Wave"), inherited across A-106, A-107, B-201,
B-221, C-305, C-310, and Book 1 Ch12. This was a real inconsistency,
not a style choice — Book 1 Ch12 explicitly and repeatedly denies
force-carrier mechanisms as fundamental, while this node's own symbol
called its output a "Force." Renamed to R_OW ("Response, One-Wave"),
which now matches this node's actual title. The mechanism itself is
unchanged — only the symbol and the word "Force" as a category name
were corrected, everywhere they appeared as an affirmed One-Wave
mechanism. Standard Model "force" and "force-carrier" language in
Book 1's Gray sections and explicit rejection passages (e.g. Ch1, Ch2,
Ch7 — "no force-carrier needed") was left untouched, since that usage
is either accurately describing the comparison framework or correctly
naming the thing being denied, not affirming a One-Wave force.

DEPENDENCIES:
	UPSTREAM: A-104 Gradient
	DOWNSTREAM: A-106 Pressure Response, E-501 Flowback

1. PURPOSE
	Converts gradient structure into field response.

	INPUT:
		∇ψ = Field Gradient

	OUTPUT:
		R_OW = Restoring Response

	TRANSFORMATION:
		∇ψ → R_OW = -A(∇ψ)

2. DEFINITION
	Restoring Response is the field reaction to gradient imbalance.

	A-104 defines:
		∇ψ = Field difference structure.

	A-105 defines:
		Resulting opposing response.

	GENERAL FORM:
		R_OW = -A(∇ψ)

	SPECIAL CASE:
		R_OW = -α∇ψ

	WHERE:
		R_OW = One-Wave restoring response.
		A = Response operator.
		α = Constant response coefficient.

	LIMIT:
		α assumes fixed response.
		A allows variable response behavior.

3. STATE VARIABLES
	INPUT:
		∇ψ

	RESPONSE:
		R_OW

	OPERATOR:
		A

	CHANNEL EXTENSIONS:
		Compression/Density.
		Expression/Phase.
		Projection.

	STATUS:
		Open.

4. MATHEMATICS
	GENERAL:
		R_OW = -A(∇ψ)

	LINEAR CASE:
		A(∇ψ)=α∇ψ

	THEREFORE:
		R_OW=-α∇ψ

	ZERO:
		∇ψ=0 → R_OW=0

	NONZERO:
		∇ψ≠0 → R_OW≠0 if A(∇ψ)≠0

	SIGN:
		Negative represents opposing response direction.
