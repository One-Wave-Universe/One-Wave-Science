## Gray — Standard Model Reference

In neuroscience, memory is understood as synaptic modification.
Long-term potentiation (LTP): repeated activation of a synapse strengthens it.
The Hebbian principle: neurons that fire together wire together.
Memory types: declarative (explicit — facts, events), procedural (implicit — skills, habits).
Short-term memory: held in prefrontal cortex, capacity ~7 items, duration ~30 seconds.
Long-term memory: consolidated during sleep, stored across distributed cortical networks.
The hippocampus is critical for forming new declarative memories.
Retrieval: memory is reconstructive, not playback — each recall partially rebuilds the memory.

In computer science, memory is storage:
RAM: fast, volatile, current working state.
Disk: slow, persistent, long-term storage.
Compression: storing patterns more efficiently than raw data.
Cache: fast local copy of frequently used data.

Standard Model strength: the synaptic model of memory is well supported.
Standard Model limitation: the synaptic model does not explain
how distributed patterns become unified experience.
How compressed patterns are retrieved and reconstructed is not fully understood.
The relationship between memory and identity is not explained.
