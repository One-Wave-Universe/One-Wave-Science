Book 3 — Scope and Status

Scope

Book 3 covers the human-biological scale in B-220's Micro←Small←Medium←Large←Macro hierarchy — organisms and physiology, one level up from Book 2's cellular scale.

Honest Status
This book does not yet exist as chapters. What follows is a scope statement and a list of what would need to be built first — not a placeholder pretending otherwise, and not invented physics content to fill pages.

What Would Need to Exist First

Real physiology — nervous system function, organ systems, homeostasis beyond the cellular ATP model already in Book 2 — has essentially no grounding nodes yet. This is the weakest-supported book in the whole series and may need new appendix-level work before it can be written honestly, or a deliberately narrower opening scope than "organisms and physiology" broadly.

Why No Content Yet

The governing rule for this book series is that appendices define the language and books apply it — no book chapter should introduce a mechanism that doesn't already exist as a node. Writing chapters for this book before that grounding exists would mean inventing physics-analog content at the book layer, which breaks that rule and would not be honest to the framework's own discipline.
