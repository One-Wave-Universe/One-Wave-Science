# Chapter 4 — Pre-Claude Architecture Recovery

Status: ACTIVE HYPOTHESIS SOURCE RECOVERY

This chapter preserves the functional architecture developed before the
cross-platform packet audit. It is not filed as a set of eighteen new
canonical nodes. It remains active research material organized by claim.

## Six Duplex Gates

```text
1 Field / present input
<-> 2 Choice / polarity
<-> 3 Move / candidate action
<-> 4(0) Mirror / null / phase interaction
<-> 5 State and scale report
<-> 6 Commit and whole-system movement
```

Every handoff is simultaneously bidirectional. Forward carries proposal,
movement, or result. Reverse carries reference, constraint, resistance,
and correction.

Each handoff is proposed to contain:

1. Open / Displace
2. Meet / Weigh
3. Shift / Settle

## Gate 4 Four Views

- Inward: compress, contain, inspect, preserve.
- Outward: express, release, communicate, propagate.
- Across: compare, measure, oppose, balance.
- Over: bridge, coordinate, modulate, oversee.

## Gate 5 and Gate 6 Closure

Working rule:

```text
loop release = completed Gate 5 state AND matching Gate 6 acceptance
```

Neither candidate generation nor administrative control can force
irreversible closure alone.

## Two Grounds

- Reference Ground: committed identity, rules, stable memory, prior state.
- Working Ground: simulations, candidates, unresolved changes.

Only accepted closure promotes Working Ground into Reference Ground.

## Meet-in-the-Middle

Two systems connect by matching enough speed, intensity, timing,
direction, and phase to form a mutually authorized shared access line.
They do not need to become identical.

## Maximum Throttle

12:1 remains a proposed ceiling. It is not derived. At the ceiling the
system should stop amplification and return toward synchronization,
redistribution, slowdown, or hold.

## Gate 7 Question

Candidate interpretation: Gate 7 is the completed shared loop that
modulates the next cycle and becomes the next Gate 1 field. This remains
an open interpretation rather than a locked sequential seventh command.
