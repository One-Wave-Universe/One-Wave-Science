D-405: Harmonic Shell

Geometric rule

2πR = nλ
R_n = nλ/(2π)
ΔR = λ/(2π)

The model distinction

Variable radius, fixed wavelength: n counts copies of the same wavelength around larger paths.
Fixed radius, variable wavenumber: n changes k and can create a dispersion-based spectrum.
For current D-405 geometry:
k_n = n/R_n = 2π/λ

So k does not change with n. The node currently quantizes shell-center geometry, not energy levels.
