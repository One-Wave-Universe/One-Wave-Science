Definition

The state in which reliable restoration of the Paired Loop is no longer possible — not the end of the relationship, but a threshold event with several possible outcomes: Separation, Recursive collapse, Access Line formation, Reset, or Dormant-but-recoverable.

Mathematics

qₙ ≤ q_break ⇒ Break candidate Hysteresis: q_break < q_return (prevents one-step toggling) Break is a function of integrity q, NOT activation a — low a with high q is rest, not break.
Integrity q oscillating with hysteresis: the system must recover above q_return before Return is accepted, even after briefly dipping below q_break.

Math Trail (dependency chain)

This is one of the nodes actually promoted from GREEN/YELLOW-mixed to full YELLOW this cycle: the break criterion is now mathematically disambiguated from energy level, with an explicit hysteresis requirement added where none existed before.

Recursive Reframing

The hysteresis gap is already a genuinely recursive, stateful mechanism — qₙ₊₁ depends on both qₙ and which side of the gap the system last crossed from, not just the instantaneous value. This is a real, working recursive rule already, not a proposal.

Yellow Audit — what's still open

q_break and q_return require scale-specific calibration. Outcome-selection math (which of the five outcomes fires) remains incomplete. Time allowed below q_break before irreversible separation is unresolved.
