E-527: Threshold-Triggered Relaxation Oscillator Model

Appendix E · energy-gated synchronization and resource cycling

The correction

Disproved as sufficient: C(t)=R(t)U(t) does not create a cycle by itself. Under constant supply it approaches a plateau. Finite substrate without recharge creates one pulse, not a repeating oscillator.

The product is a readout of coordinated available response. A genuine repeating relaxation cycle requires recharge, state-dependent depletion, and separate on/off thresholds.

Reduced model

C(t) = R* U(t) dU/dt = P - λU - D h U h: 0 → 1 when C ≥ C_on h: 1 → 0 when C ≤ C_off 0 < C_off < C_on

R* is the settled synchronization level supplied by the E-524 layer. U is stored resource and h is the release state.

Exact cycle conditions

U_charge = P/λ U_release = P/(λ+D) U_on = C_on/R* U_off = C_off/R* Cycle iff: P/λ > C_on/R* and P/(λ+D) < C_off/R*

Exact predicted period

T_charge = (1/λ) ln[(U_charge-U_off)/(U_charge-U_on)] T_release = (1/(λ+D)) ln[(U_on-U_release)/(U_off-U_release)] T_period = T_charge + T_release

Simulation cases

Case 1. Constant supply: the resource and C=R*U approach fixed plateaus. No cycle.
Case 2. A finite unrecharged substrate creates one rise and fall. This is a pulse, not an oscillator.
Case 3. Recharge, release-dependent depletion, and hysteresis produce repeated cycles.
Analytic parameter map. A cycle exists only where both charging and discharging inequalities are satisfied.
