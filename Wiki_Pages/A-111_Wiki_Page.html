Definition

Recursion is a closed update cycle where the output of one step becomes the input of the next. Memory alone is not recursion. Recursion requires both memory and a feedback rule [1].

ψn+1=f(ψn,ψn−1) \psi_{n+1} = f(\psi_n, \psi_{n-1})

Memory+Feedback Rule⇒Recursion \text{Memory} + \text{Feedback Rule} \Rightarrow \text{Recursion}

Mathematics

Full One-Wave update rule:

ψin+1=ψin+(1−γ)(ψin−ψin−1)+βi(⟨ψjn⟩−ψin) \psi_i^{n+1} = \psi_i^n + (1-\gamma)(\psi_i^n - \psi_i^{n-1}) + \beta_i(\langle \psi_j^n \rangle - \psi_i^n)

Closed update cycle:

fk(ψ)=ψfor integer k f^k(\psi) = \psi \quad \text{for integer } k

Locality Rule

Nearest-neighbor coupling is confirmed through βi(⟨ψjn⟩−ψin)\beta_i(\langle \psi_j^n \rangle - \psi_i^n). Whether additional nonlocal effects beyond nearest-neighbor exist remains open [1].

Oscillation is not required for recursion [1].

Operational Chain

Oscillation+Memory+Feedback Rule⇒Recursion⇒Persistent Mode \text{Oscillation} + \text{Memory} + \text{Feedback Rule} \Rightarrow \text{Recursion} \Rightarrow \text{Persistent Mode}

Metadata and Wave Layer

Metadata is the compressed description of a wave configuration. Wave data is metadata that has entered an active update cycle.

A wave state is represented as:

Ψ=(A,f,ϕ,x,t,m) \Psi = (A, f, \phi, x, t, m)

where:

  • AA = amplitude.
  • ff = frequency.
  • ϕ\phi = phase.
  • xx = position.
  • tt = time.
  • mm = metadata state.

Recursive wave update:

Ψn+1=F(Ψn,Ψn−1,M) \Psi_{n+1} = F(\Psi_n, \Psi_{n-1}, M)

where:

  • Ψn\Psi_n = current wave state.
  • Ψn−1\Psi_{n-1} = memory state.
  • MM = metadata constraints.

Metadata describes the state. Wave data is metadata expressed as an active update process. Conversion requires state identity, coupling rule, timing relationship, and propagation behavior.

Interference Layer

ψ(t)=Asin(2πft+ϕ) \psi(t) = A\sin(2\pi f t + \phi)

Multiple waves combine as:

Ψ(t)=∑k=1NAksin(2πfkt+ϕk) \Psi(t) = \sum_{k=1}^{N} A_k\sin(2\pi f_k t + \phi_k)

Measure reinforcement, cancellation, harmonic overlap, and phase alignment.

fb=|f1−f2| f_b = |f_1 - f_2|

Harmonic Mapping

fn=f0(2n/12) f_n = f_0(2^{n/12})

Track root distance, octave position, interval ratio, and oscillation width.

Relative frequency:

R=f2f1 R = \frac{f_2}{f_1}

Stability Measurement Layer

Persistence is not defined as lack of movement. A persistent pattern may oscillate while preserving its structure.

State Difference

Dn=∥Ψn−Ψn−1∥ D_n = \|\Psi_n - \Psi_{n-1}\|

Used to measure local change.

Structural Stability

SR=1−|Rn−Rn−1|Rmax S_R = 1 - \frac{|R_n - R_{n-1}|}{R_{max}}

where:

  • RnR_n = relationship state at time nn.
  • RmaxR_{max} = maximum allowed relational change.

A wave remains stable when its internal relationships remain consistent.

Combined Stability Score

Stotal=wRSR+wfSf+wϕSϕ+wASA S_{total} = w_R S_R + w_f S_f + w_\phi S_\phi + w_A S_A

where:

  • SRS_R = relational stability.
  • SfS_f = frequency stability.
  • SϕS_\phi = phase stability.
  • SAS_A = amplitude stability.
  • ww = weighting factors.

Persistence Test

A recursive wave enters persistent mode when:

∥Ψn+k−Ψn∥<ϵ \|\Psi_{n+k} - \Psi_n\| < \epsilon

for a defined tolerance.

Possible outcomes:

A recursive system is stable when the wave state remains measurable, bounded, and repeatable under its own update rule.
