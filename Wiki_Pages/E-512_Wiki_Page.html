Definition

The signed pair of positions (compression-side, expression-side) a chord produces once Chord Rotation has re-centered it -- a result value, not a function or coordinate system itself.

Mathematics

Window=(P_compress,P_expr). A Major=(−5,+4). A Minor=(−5,+3) -- both corrected this session from an earlier backwards version.
Same clock figure -- the Oscillation Window is the (-5,+4) pair itself, read off after rotation.

Math Trail

The fifth (compression side) appears to stay fixed at -5 across Major/Minor while the third (expression side) shifts by exactly one semitone -- a real, if thin, observed pattern from only two data points.

Recursive Reframing

Not itself recursive -- this is a per-chord snapshot result. A recursive extension would track the Window across a chord progression over time, explicitly named as unresolved.

Yellow Audit

Only 2 data points exist -- nowhere near enough to call the pattern a rule. No downstream consumer defined yet beyond E-513. Explicitly NOT the same concept as B-208 Threshold Windows despite the shared name -- flagged to prevent a repeat of the Mirror/Mirror-Gate naming confusion.
